
# IEG CRM 사용자 및 설치 매뉴얼

본 문서는 Gemini AI 기반 영업 관리 시스템(CRM)의 설치 및 사용 방법을 설명합니다.

---

## 1. 시스템 개요

IEG CRM은 영업 활동을 효율적으로 관리하기 위한 웹 애플리케이션입니다. 구글 Gemini AI를 활용하여 거래처 등급 분석, 이메일 초안 작성, 회의록 요약 등의 기능을 제공합니다.

### 주요 기능
1.  **대시보드**: 개인/팀/부서별 목표, 실적, 예정, 미정 금액 분석 및 시각화.
2.  **AI 거래처 등급 산정**: 내부 데이터와 외부 뉴스/재무 정보를 Gemini가 검색하여 S~D 등급으로 자동 평가.
3.  **파이프라인 관리**: 영업 단계별 진행 상황을 칸반 보드 형태로 시각화.
4.  **주간 리포트**: 파이프라인 현황을 분석하여 금주 우선순위 업무를 이메일 형태로 제안.
5.  **데이터 내보내기/불러오기**: 구글 시트와 양방향 데이터 동기화 지원.

## 2. 설치 및 배포 (Google Apps Script 환경)

### A. 구글 스프레드시트 설정
1.  새 스프레드시트를 생성하거나 기존 시트를 엽니다. (ID: `1fWeLcQvotKZGgxelAOvKt_UXyYvdoneYVBeVXFMLRKY` 사용 중)
2.  상단 메뉴의 **[확장 프로그램]** > **[Apps Script]**를 클릭하여 스크립트 에디터를 엽니다.

### B. 백엔드 코드 설정 (Code.gs)
1.  `Code.gs` 파일의 기존 내용을 모두 지웁니다.
2.  제공된 `backend/Code.js`의 내용을 복사하여 붙여넣고 저장(Ctrl+S)합니다.

### C. 프론트엔드 코드 설정 (index.html)
1.  Apps Script 에디터 좌측의 **[+] (파일 추가)** 버튼을 클릭하고 **[HTML]**을 선택합니다.
2.  파일 이름을 **`index`**로 입력합니다. (자동으로 `index.html`이 됩니다.)
3.  생성된 `index.html`의 내용을 지우고, **React 앱을 빌드한 결과물**을 붙여넣습니다.
    *   *참고: 빌드 환경이 없다면 아래 예시 코드를 사용하여 테스트할 수 있습니다.*

### D. 권한 승인 및 실행
1.  상단 툴바의 함수 선택 목록에서 **`getCRMData`**를 선택합니다.
2.  **[실행]** 버튼을 클릭합니다.
3.  "권한 검토" 팝업이 뜨면 **[권한 검토]** > 계정 선택 > **[고급]** > **[...으로 이동 (안전하지 않음)]** > **[허용]**을 차례로 클릭합니다.

### E. 웹 앱 배포
1.  우측 상단의 **[배포]** > **[새 배포]**를 클릭합니다.
2.  유형 선택 옆의 톱니바퀴 > **[웹 앱]**을 선택합니다.
3.  **다음 사용자로서 실행**: '나(웹 앱 소유자)'
4.  **액세스 권한**: '모든 사용자' (또는 조직 내 사용자)
5.  **[배포]** 클릭 후 생성된 **웹 앱 URL**을 클릭하여 CRM에 접속합니다.

## 3. 사용 방법

### A. 대시보드 활용
*   **범위 선택**: 상단의 [개인별 | 팀별 | 전체] 버튼을 눌러 데이터를 필터링합니다. 개인별 선택 시 우측 드롭다운에서 담당자를 변경할 수 있습니다.
*   **지표 확인**:
    *   **목표 금액**: 선택된 담당자/팀의 연간(또는 월간) 목표 합계.
    *   **매출 실적**: '계약 성사(Closed Won)' 단계의 합계.
*   **내보내기**: 우측 상단 [내보내기] 버튼을 눌러 구글 시트로 데이터를 저장하거나 PDF로 출력합니다.

### B. 거래처 관리 및 AI 등급 분석
1.  좌측 메뉴의 **[거래처 및 매출 관리]**로 이동합니다.
2.  리스트의 **[수정]** 아이콘을 눌러 정보를 수정하거나, **[등록]** 버튼으로 새 거래처를 추가합니다.
3.  주소 입력 시 [주소 검색] 버튼을 활용하여 간편하게 입력할 수 있습니다.
4.  거래처 상세 화면에서 **[등급 분석 실행]**을 누르면 Gemini AI가 외부 데이터를 검색하여 등급을 산정합니다.

### C. 설정 (직원 관리 및 데이터 초기화)
1.  **직원 관리**: 영업 담당자를 추가, 수정, 삭제할 수 있으며 프로필 사진을 업로드할 수 있습니다.
2.  **샘플 데이터 업로드**: [구글 시트에 샘플 업로드] 버튼을 누르면, 화면에 보이는 테스트 데이터를 즉시 구글 시트에 저장합니다.

---

## 4. 주의 사항
*   **"index 파일이 없습니다" 에러**: 2-C 단계를 수행하여 `index.html` 파일을 반드시 생성해야 합니다.
*   **데이터가 시트에 저장되지 않음**: 앱 상단의 [저장] 버튼을 누르거나 설정 메뉴의 업로드 기능을 사용해야 실제 시트에 반영됩니다.
