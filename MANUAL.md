# IEG CRM 사용자 및 설치 매뉴얼

본 문서는 Gemini AI 기반 영업 관리 시스템(CRM)의 설치 및 사용 방법을 설명합니다.

---

## 1. 시스템 개요

IEG CRM은 영업 활동을 효율적으로 관리하기 위한 웹 애플리케이션입니다. 구글 Gemini AI를 활용하여 거래처 등급 분석, 이메일 초안 작성, 회의록 요약 등의 기능을 제공합니다.

### 주요 기능
1.  **대시보드**: 개인/팀/부서별 목표, 실적, 예정, 미정 금액 분석 및 시각화.
2.  **AI 거래처 등급 산정**: 내부 데이터와 외부 뉴스/재무 정보를 Gemini가 검색하여 S~D 등급으로 자동 평가.
3.  **파이프라인 관리**: 영업 단계별 진행 상황을 칸반 보드 형태로 시각화.
4.  **주간 리포트**: 파이프라인 현황을 분석하여 금주 우선순위 업무를 이메일 형태로 제안.
5.  **데이터 내보내기**: JSON 형태로 데이터 백업 가능.

---

## 2. 설치 및 배포 (Google Drive / Apps Script 환경)

본 시스템은 React 기반의 SPA(Single Page Application)로 제작되었습니다. 구글 드라이브(Google Apps Script)에 배포하여 사내 인트라넷처럼 사용할 수 있습니다.

### 사전 준비
1.  Google Cloud Console에서 프로젝트 생성.
2.  **Gemini API Key** 발급 (AI Studio).
3.  Google Apps Script 프로젝트 생성.

### 배포 단계
1.  **빌드**: `npm run build` 명령어를 통해 React 앱을 단일 HTML 파일로 번들링합니다. (또는 호스팅 서버에 배포)
2.  **Google Apps Script (GAS) 설정**:
    *   `code.gs` 파일을 생성하고 `doGet()` 함수를 작성하여 HTML을 서빙하도록 합니다.
    *   스프레드시트와 연동하려면 GAS 내에서 `SpreadsheetApp` 클래스를 사용하는 백엔드 함수(`getContacts`, `saveContact` 등)를 작성합니다.
3.  **환경 변수**:
    *   Gemini API Key는 GAS의 '스크립트 속성'에 저장하여 보안을 유지합니다.

## 3. 사용 방법

### A. 대시보드 활용
*   **범위 선택**: 상단의 [개인별 | 팀별 | 부서별] 버튼을 눌러 데이터를 필터링합니다.
*   **지표 확인**:
    *   **목표 금액**: 사전에 설정된 거래처별 목표의 합계.
    *   **매출 실적**: '계약 성사(Closed Won)' 단계의 합계.
    *   **매출 예정**: 계약 확률 60% 이상인 건들의 합계.
    *   **매출 미정**: 계약 확률 60% 미만인 건들의 합계.
*   **내보내기**: 우측 상단 [내보내기] 버튼을 눌러 현재 데이터를 파일로 저장합니다.

### B. 거래처 관리 및 AI 등급 분석
1.  좌측 메뉴의 **[거래처 관리]**로 이동합니다.
2.  거래처를 클릭하여 상세 페이지로 진입합니다.
3.  **'AI 거래처 등급 평가'** 카드에서 **[등급 분석 실행]** 버튼을 클릭합니다.
    *   AI가 구글 검색(Google Search)을 통해 해당 기업의 최신 뉴스, 재무 상태를 파악하고 내부 메모와 결합하여 등급(S, A, B, C, D)을 산정합니다.

### C. 주간 리포트 생성
1.  좌측 사이드바 하단의 **[주간 리포트 생성]** 버튼을 클릭합니다.
2.  AI가 현재 파이프라인의 정체 구간과 마감 임박 건을 분석합니다.
3.  우선 방문 대상과 해야 할 일이 정리된 리포트가 생성됩니다.

### D. Gemini AI 도구
*   **이메일 작성**: 거래처 상세 화면 -> 'AI 이메일 초안' 탭에서 상황만 입력하면 격식 있는 메일을 작성해줍니다.
*   **회의록 요약**: '회의록 요약' 탭에서 거친 메모를 붙여넣으면 액션 아이템을 도출합니다.

## 4. 주의 사항
*   AI 분석 결과는 참고용이며, 최종 판단은 영업 담당자가 수행해야 합니다.
*   외부 데이터 검색(Grounding) 기능 사용 시 약간의 시간이 소요될 수 있습니다.
*   금액 단위는 원화(₩)이며, 1원 단위까지 표기됩니다.
